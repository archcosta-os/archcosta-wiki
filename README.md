# ArchCosta Wiki

Welcome to the documentation for **ArchCosta OS**.

## Introduction
ArchCosta is a professional, performance-tuned distribution based on Arch Linux. It is designed for low-end systems while providing a feature-rich, "pro" environment inspired by Archcraft and CachyOS.

## Key Features
- **Performance Kernel:** Uses `linux-cachyos` with optimized scheduler.
- **Efficient Desktop:** Customized XFCE4 environment.
- **Modern Launchers:** Pre-configured Rofi and Dunst.
- **Low RAM Mode:** Custom scripts to toggle compositing and heavy services.
- **Optimized Browser:** Brave browser with performance flags.
